My Projects

Here are some of the projects I have worked on.

Chatbot for Disease Prediction and Ayurvedic Treatment Recommendation

This chatbot was developed as my final-year project to combine the power of modern technology with traditional healthcare practices. The chatbot utilizes machine learning algorithms, specifically the Random Forest model, to predict diseases based on user symptoms. Additionally, it provides Ayurvedic treatment recommendations tailored to the user's input.

Key Features:

  • Interactive and user-friendly chatbot interface
  • Accurate disease prediction using Ayurvedic data and machine learning
  • Seamless integration of modern predictive tools with traditional medicine

Technologies: Python, Machine Learning (Random Forest)

Price Tracking Web Application

This web application was created as a minor project to address the challenge of price fluctuations in e-commerce. It allows users to track and compare product prices across multiple platforms, helping them make informed purchasing decisions. Web scraping techniques were used to extract data dynamically from e-commerce websites.

Key Features:

  • Real-time price tracking from various e-commerce platforms
  • Product sorting and filtering options for better comparisons
  • User-friendly interface for seamless navigation and interaction

Technologies: Python, BeautifulSoup4

Bitcoin Click Game

A fun and engaging Bitcoin-themed clicker game, this project showcases my creativity and frontend development skills. The game challenges players to collect as many Bitcoins as possible through rapid clicks, featuring an interactive interface and dynamic score tracking.

Key Features:

  • Interactive design with dynamic animations
  • Real-time score tracking and leaderboard
  • Optimized for smooth performance on various devices

Technologies: HTML, CSS, JavaScript
